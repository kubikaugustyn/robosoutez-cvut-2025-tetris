# Robosoutěž FEL ČVUT 2025 - Tetris
Program(y?) pro robota LEGO® MINDSTORMS® pro řešení soutěžní úlohy Tetris v Robosoutěži FEL ČVUT 2025.

Pro programování EV3 kostky používám (ev3-micropython)[https://pybricks.com/ev3-micropython/index.html].
